SlidyCat

Minimal design, minimal mark-up slideshow.

Dynamic.

This slideshow does the heavy lifting for you. SlidyCat will automatically create the mark-up required for the slideshow. All you need to do is specify the images.

Optionally, you can specify links that you would like the images to direct to.

Friendly.

The outputted mark-up is designer friendly; it makes use of HTML in a sensible and ease-to-understand way, as well as simple class names for maximum flexibility.

Simple.

Easily add this slideshow to your website with little mark-up change or coding experience.

Example

The above slideshow was created with the following lines of HTML and Javascript:

<div id="example"></div>
…
<script>
$("#example").slidyCat({
	  images: {"slideshow_1.jpg", "slideshow_2.jpg", "slideshow_3.jpg"},
	   links: {"http://www.google.ca/", "", "http://www.reddit.com/r/frontend"},
	controls: true,
	slidemap: true,
	autoplay: true
});
</script>
